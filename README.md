# Agent Economy OS

Experimentation surface for **agent-economy** workflows inside the shared pnpm workspace pattern (`agent-economy` + `api-server` + `mockup-sandbox`).

## 💼 Real-World Use Case
Product and research teams mocking how autonomous agents, queues, and dashboards could coordinate—before hardening billing, identity, and policy.

## 💰 Potential Value
- Useful as an **internal prototype** to test UX for agent orchestration or “economy” narratives without committing to a vendor stack.
- Shares the same disciplined monorepo layout as sibling repos for faster onboarding if the thesis narrows to a concrete buyer.

## 🚀 Demo / Access
Self-hosted / local only. Install pnpm, run workspace `typecheck`/`build` scripts, and lift `artifacts/api-server` with `.env` from sibling README patterns. No public demo URL is published for this repository.
